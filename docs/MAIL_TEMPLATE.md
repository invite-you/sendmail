## 검증

- 존재 안 하는 토큰 → 오류
- {컬럼} 토큰은 대소문자 구분 없이 매칭
- 첨부 >10MB → 오류

## 중복 이메일 처리

- 표 렌더링을 위해 템플릿 엔진 사용 (예: Python Ninja 2)
